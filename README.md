## 🖥️ What it does

fastvideosegmenter is a desktop app for Windows that splits videos into smaller segments with ffmpeg stream copy.

It keeps the original video and audio streams intact, so it avoids re-encoding. That helps you cut videos fast and keep quality the same.

Use it when you want to:

- Split long videos into parts
- Cut clips from a larger file
- Keep the same video quality
- Process several files with less waiting
- Work in a dark mode app on your PC

## 🔍 What makes it fast

fastvideosegmenter uses stream copy through ffmpeg.

That means it copies the video and audio data instead of building a new encode. This keeps the process quick and helps preserve quality.

It also uses parallel processing, which can help when you work with more than one segment or file.
